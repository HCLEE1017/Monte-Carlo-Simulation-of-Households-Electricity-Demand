Monte Carlo Simulation of Household Electricity Demand

***Guideline of Upload File***

File Name Requirement

The system only accepts "csv" file and each file can only contain the data of one device, but allows different Household Id. Please follow the guideline to save your file name.

For Example, if the file content is about the households' electricity consumption of TV, Please save the file name as "TV.csv". (Please make sure the name is in upper case) Here is the list for other devices...

For electricity consumption of TV: "TV.csv"

For electricity consumption of Kettle: "KT.csv"

For electricity consumption of Microwave: "MICRO.csv"

For electricity consumption of Toaster: "TER.csv"

For electricity consumption of Washing Machine: "WM.csv"

For electricity consumption of Tumble Dryer: "TD.csv"

For electricity consumption of Dishwasher: "DW.csv"

For electricity consumption of Fridge: "FRI.csv"

For electricity consumption of Freezer: "FRE.csv"

*Note: The file of Fridge-Freezer could be save as either "FRI" or "FRE"

File Content Requirement

Example of display

There are only four columns that you must provide...

    Household_Id: you can decide any household id
    Date (yyyy/mm/dd): the day that the electricity consumption has been recorded
    Time (hh:mm): the time that the electricity consumption has been recorded (Please provide the time from 00:00 to 23:59)
    Watt: recorded electricity consumption in every minute
